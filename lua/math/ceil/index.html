Lua's math.ceil in TypeScript

✓ Verified: Lua 5.4
Examples tested against actual runtime. CI re-verifies continuously. Only documented examples are tested.
Rosetta Stone: python/ceil · php/ceil · perl/ceil · r/ceiling · julia/ceil · elixir/ceil · clojure/ceil

How to use

Install via yarn add locutus and import: import { ceil } from 'locutus/lua/math/ceil'.

Or with CommonJS: const { ceil } = require('locutus/lua/math/ceil')

Use a bundler that supports tree-shaking so you only ship the functions you actually use. Vite, webpack, Rollup, and Parcel all handle this. For server-side use this is less of a concern.
