xslt_backend_name

(PHP 4 >= 4.3.0)

xslt_backend_name — Returns the name of the backend

Description

string xslt_backend_name ( void )

xslt_backend_name() gets the name of the backend.

Return Values

Returns Sablotron.

Examples

Example #1 xslt_backend_name() example

<?php

echo xslt_backend_name(); // Sablotron

?>
