OCI-Lob->close — Closes LOB descriptor

Description

OCI-Lob
bool close ( void )

Closes descriptor of LOB or FILE. This function should be used only with OCI-Lob->writeTemporary.

Return Values

Returns TRUE on success or FALSE on failure.
