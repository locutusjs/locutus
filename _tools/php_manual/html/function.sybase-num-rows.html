sybase_num_rows

(PHP 4, PHP 5)

sybase_num_rows — Get number of rows in a result set

Description

int sybase_num_rows ( resource $result )

sybase_num_rows() returns the number of rows in a result set.

Parameters

result

Return Values

Returns the number of rows as an integer.
