ingres_num_fields

(PHP 4 >= 4.0.2, PHP 5 <= 5.0.5, PECL ingres:1.0-1.4.3)

ingres_num_fields — Get the number of fields returned by the last query

Description

int ingres_num_fields ([ resource $link ] )

ingres_num_fields() returns the number of fields in the results returned by the Ingres server after a call to ingres_query()

Parameters

link

The connection link identifier. If not specified, the last opened link is used.

Return Values

Returns the number of fields.
