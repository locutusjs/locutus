acosh

(PHP 4 >= 4.0.7, PHP 5)

acosh — Inverse hyperbolic cosine

Description

float acosh ( float $arg )

Returns the inverse hyperbolic cosine of arg , i.e. the value whose hyperbolic cosine is arg .

Parameters

arg

The value to process

Return Values

The inverse hyperbolic cosine of arg
