tidyNode::getParent

(PHP 5 >= 5.2.2)

tidyNode::getParent — returns the parent node of the current node

Description

tidyNode tidyNode::getParent ( void )

Returns the parent node of the current node.

Return Values

Returns a tidyNode if the node has a parent, or NULL otherwise.
