gethostbynamel

(PHP 4, PHP 5)

gethostbynamel — Get a list of IP addresses corresponding to a given Internet host name

Description

array gethostbynamel ( string $hostname )

Returns a list of IP addresses to which the Internet host specified by hostname resolves.

Parameters

hostname

The host name.

Return Values

Returns an array of IP addresses or FALSE if hostname could not be resolved.

Examples

Example #1 gethostbynamel() example

<?php
$hosts = gethostbynamel('www.example.com');
print_r($hosts);
?>

The above example will output:

Array
(
    [0] => 192.0.34.166
)
