libxml_clear_errors

(PHP 5 >= 5.1.0)

libxml_clear_errors — Clear libxml error buffer

Description

void libxml_clear_errors ( void )

libxml_clear_errors() clears the libxml error buffer.

Return Values

No value is returned.
