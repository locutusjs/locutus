PHP's var_dump in TypeScript

How to use

Install via yarn add locutus and import: import { var_dump } from 'locutus/php/var/var_dump'.

Or with CommonJS: const { var_dump } = require('locutus/php/var/var_dump')

Use a bundler that supports tree-shaking so you only ship the functions you actually use. Vite, webpack, Rollup, and Parcel all handle this. For server-side use this is less of a concern.

Examples

These examples are extracted from test cases that automatically verify our functions against their native counterparts.

#codeexpected result

Notes

  • For returning a string, use var_export() with the second argument set to true

Dependencies

This function uses the following Locutus functions:

  • _phpRuntimeState (php/_helpers)
  • _phpTypes (php/_helpers)
  • echo (php/strings)
