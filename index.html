phpjs by kvz

Use PHP functions in JavaScript

PHP.JS is an open source project in which we try to port PHP functions to JavaScript. By including the PHP.JS library in your own projects, you can use your favorite PHP functions client-side.

Install

No server component required. To use PHP.JS you can either: * Download the default package * Customize & download a package (only select what you need) * Goto one of the Functions and just Copy-Paste it More info at http://phpjs.org
